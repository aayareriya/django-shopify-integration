# Django-Shopify Integration

A simple Django app that connects with Shopify to receive new orders automatically using webhooks. Includes ngrok for local testing and a clean Bootstrap UI to view orders.

## Features

- Receive Shopify order creation events via webhooks  
- Save order details (customer name, total price, currency, order date) in Django database  
- Use ngrok to expose local server for webhook testing  
- Clean and responsive Bootstrap-based UI
